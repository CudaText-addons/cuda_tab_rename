Plugin for CudaText.
Allows to rename tabs in the editor.

Adds menu item "Rename tab" to editor tab context menu. 
And commands to rename tab and reset name to main menu in:
	"Plugins > Tab-Rename" 

-----------------------------

Prefix for tabs with altered names can be changed in config, accessible via menu:
	"Options > Settings-Plugins > Tab-Rename"
